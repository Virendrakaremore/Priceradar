# Priceradar

## Project Description

Priceradar is a web-based application that helps users track and compare product prices efficiently using modern web technologies.

##  Tech Stack

* Frontend: React.js
* Backend: Python (Flask / FastAPI)
* API Communication: REST APIs

##  Project Structure

```
Priceradar/
│
├── Frontend/   # UI of the application
├── Backend/    # Server and API logic
```

##  Features

* Real-time price tracking
* User-friendly dashboard
* API-based communication between frontend and backend

##  Deployment

* Frontend deployed on Vercel
* Backend deployed on Render / Railway
